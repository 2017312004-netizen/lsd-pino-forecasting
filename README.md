# LSD-PINO Forecasting

Physics-Informed Neural Operator framework for spatiotemporal forecasting of lumpy skin disease (LSD) spread.

## Overview

This repository contains supplementary materials associated with the manuscript:

**"Physics-Informed Neural Operator Framework for Spatiotemporal Forecasting of Lumpy Skin Disease Spread"**

The proposed framework integrates:

* Data assimilation for ADR model calibration
* ADR-based spatiotemporal outbreak simulation
* Ensemble generation through parameter perturbation
* Neural surrogate modeling using CNN, FNO, and PINO
* Physics-informed forecasting of lumpy skin disease spread

The objective of this study is to investigate the applicability of neural operator learning for forecasting livestock disease outbreaks under limited observational data conditions.

### Supplementary Video S2: Forecasting Results

Comparison between reference outbreak evolution and surrogate-model predictions at a 14-day forecasting horizon.

The forecasting experiments include:

* Convolutional Neural Network (CNN)
* Fourier Neural Operator (FNO)
* Physics-Informed Neural Operator (PINO)

Each animation contains:

* Ground Truth (GT)
* Model Prediction
* Prediction Error (Prediction − GT)
